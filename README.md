#  Student Spark – Real-Time Idea Sharing Platform
## 🌐 Live Demo

🔗 https://jagadish3723.github.io/ideas-sharing-platform/

> A scalable full-stack web application enabling users to share ideas, collaborate, and engage in community-driven discussions.

---

## 🌟 Overview

**Student Spark** is a community-driven platform where users can:
- 💡 Share innovative ideas  
- 💬 Discuss and collaborate with others  
- 👍 Engage through comments and upvotes  

The project demonstrates **real-world product development**, focusing on scalability, performance, and user interaction.

---

## ⚙️ Tech Stack

**Frontend**
- HTML, CSS, JavaScript  

**Backend**
- Node.js, Express.js  

**Database**
- MongoDB  

**Other Tools & Concepts**
- REST APIs  
- JWT Authentication  
- bcrypt (Password Hashing)  
- Role-Based Access Control  
- Deployment (Render, GitHub Pages)  

---

## 💼 Key Achievements

- 🚀 Built a **production-ready MERN stack application** supporting idea sharing and discussions, onboarding **20+ active users**.

- ⚙️ Developed **12+ RESTful APIs** for authentication, posts, comments, likes, and user management.

- 🔐 Implemented **secure authentication system** using **JWT, bcrypt, and role-based authorization**, securing critical endpoints.

- ⚡ Optimized performance using **MongoDB indexing and schema design**, reducing API response time by **30–40%**.

- 🌐 Managed **end-to-end deployment pipeline**:
  - Backend hosted on **Render**
  - Frontend deployed via **GitHub Pages**
  - Configured environment variables & CORS policies
